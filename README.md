# woptimize

**woptimize** is a Windows optimization utility written entirely in Batch (`.bat`).
It provides a structured, menu-driven interface for applying system, CPU, memory,
network, and service optimizations ‚Äî without executing anything accidentally.

Nothing runs automatically on launch. All actions require explicit user selection
through sub-menus.
